# R6 Rewind Tools & Mods

Explore a collection of tools and mods for the **R6 Rewind Loader** — including unlock-all utilities, modding tools, and optimizations for R6.

Get the latest R6 Rewind Loader **[here:](https://github.com/creedfx/R6-Rewind/releases/latest/)**

## Included Tools

- **Cracks.zip** – Game files and additional tools
- **Heated Metal.zip** – Modding tool for Heated Metal
- **Helios Loader.zip** – Modding tool for Heated Metal
- **R6 Liberator 1.exe** – Simple modding tool for Y1S0–Y4S4
- **R6 Liberator 2.exe** – Unlock-all & playlist tool for Y5S3 Operation Shadow Legacy
- **Shears 0.2.0.zip** – Tool to reduce game size
- **Y7S2 Unlock All.zip** – Unlock-all for Y7S2 Operation Vector Glare
- **Y7S4 Unlock All.zip** – Unlock-all for Y7S4 Operation Solar Raid (Charms are not supported)

## Disclaimer

These tools are designed for different game versions and purposes.
Make sure to use the correct file for your specific season and setup.
